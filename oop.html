OOP - Object Oriented Programming

Definitions

Object - An instance of a class, each object has unique attributes and functions, object is like a cookie

Class - Template to create similar objects, class is like a cookie cutter

Constructor - Special method that initializes objects with no return type. Overloaded constructors give object unique attribute values

Subclass - Class derived from another class, has an is-a relationship with superclass

Superclass - Class that subclass is derived from. All Java Classes have are subclases of the Object class

Public Keyword - Can be used when declaring methods and/or attributes, will be visible to any foreign entity. The class and constructor are always public, methods depend on their function

Private Keyword - Can be used when declaring methods and/or attributes, can only be accessed in class. Attributes are mostly private, methods depend on their function

This Keyword - Refers to current object in a method, makes code more readable and clearer when parameters have same name
