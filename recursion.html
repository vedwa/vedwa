Recursion

Definitions

Recurse - Methods invoking themselves to perform desired function, can always be written iteratively

Iteration - Uses loops to repeat same function, can always be written recursively

Base Case - Condition that stops recursion

StackOverflowException - Unbounded recursion, typically the base case never occurs and is invalid, or doesn't exist

Infinite Loop - Loop keeps running because precondition is always evaluated as true, and never becomes false
