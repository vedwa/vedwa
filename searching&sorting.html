Searching & Sorting

Definitions

Searching - Searching through an array to find a specific value, and where the value is

Sorting - Algorithms to sort array values either ascending/descending order

Binary Search - Repeatedly divides a sorted array in half until it finds the target value, very fast

Sequential Search - Going through each element, starting from beginning, until target is found

Selection Sort - Repeatedly selects smallest/biggest element and moves it to beginning/end of array. Traversals become smaller each time

Insertion Sort - Repeatedly takes inputs from data and inserts it where it belongs until no input elements remain

Merge Sort - Keeps dividing array into halves until they become singular arrays which are guaranteed to be sortedd, then merges them together in desired order

Quick Sort - Splits data in two, then sorts and combines
